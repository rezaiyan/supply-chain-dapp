Supply chain Dapp, is an application to manage the process of an item from harvesting to purchasing by consumers
the owner of contract is able to add/remove different roles such as farmer, distributor, retailer, consumer.
All type of roles can see the items in the panel but the functionalities are limited per Role.
 for example Owner has full access, Farmer can do Harvest,Process,Pack,PutForSale.
 Distributor can do buy and ship, retailer can do receive, consumer can do purchase.


To run the application do following instructions:

cd frontend/
npm install
cd ..
npm install
npx hardhat compile
npx hardhat test
npx hardhat run ./tasks/deploy.ts --network localhost //change the network name to sepolia if you want to deploy it on testnet
npx hardhat node

Open another terminal:
cd frontend/
npm start

Note: in case you deployed to sepolia make sure to edit `.env` to set the correct variables to INFURA_PROJECT_ID and MNEMONIC
